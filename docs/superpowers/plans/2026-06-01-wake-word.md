# Wake Word ("Hey Murmur") Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add "Hey Murmur" wake-word activation using Picovoice Porcupine so users can start recording hands-free; recording stops after 3 seconds of silence (vs 1 second for hotkey mode).

**Architecture:** A new `WakeWordListener` class runs its own sounddevice stream (int16, 16kHz, 512-sample frames) in the background and calls `controller.wake_start()` on detection. `AudioCapture.start()` gains an optional `silence_blocks` param so wake-triggered recording uses 30 blocks (3s) instead of the default 10 (1s). Toggle in settings enables/disables the listener.

**Tech Stack:** pvporcupine 3.x, sounddevice, numpy, CustomTkinter (settings UI).

```python
import os
from pathlib import Path
from typing import Callable, Optional
import numpy as np
import sounddevice as sd

try:
    import pvporcupine
except ImportError:
    pvporcupine = None  # graceful degradation if not installed

MODEL_PATH = Path(os.environ.get("APPDATA", Path.home())) / "MurmurAI" / "hey_murmur.ppn"


class WakeWordListener:
    def __init__(
        self,
        access_key: str,
        model_path: str,
        on_detect: Callable[[], None],
    ) -> None:
        self._access_key = access_key
        self._model_path = model_path
        self._on_detect  = on_detect
        self._porcupine  = None
        self._stream: Optional[sd.InputStream] = None
        self._running    = False

    def start(self) -> None:
        self._porcupine = pvporcupine.create(
            access_key=self._access_key,
            keyword_paths=[self._model_path],
        )
        self._running = True
        self._stream = sd.InputStream(
            samplerate=self._porcupine.sample_rate,
            channels=1,
            dtype="int16",
            blocksize=self._porcupine.frame_length,
            callback=self._on_audio,
        )
        self._stream.start()

    def stop(self) -> None:
        self._running = False
        if self._stream:
            self._stream.stop()
            self._stream.close()
            self._stream = None
        if self._porcupine:
            self._porcupine.delete()
            self._porcupine = None

    def is_running(self) -> bool:
        return self._running

    def _on_audio(self, indata, frames, time, status) -> None:
        if not self._running or self._porcupine is None:
            return
        pcm = indata[:, 0].astype(np.int16)
        if self._porcupine.process(pcm) >= 0:
            self._on_detect()
```
